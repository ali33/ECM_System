1.     Overview

Archive product is part of ECM solution set which help user store, search and simply capture imaging document. This product has two main features below:

         Retrieving: Allow user retrieve documents that stored in system with incoming from Email, Scanner, released by Capture product

         Capturing: Provide the Light Capture screen that help user to Scan documents by scanner, import file system and capture picture or video by embed or plus in camera.

 

2.     Login

This feature provides the gate to access to system. This make sure only valid user can access system to work with appropriate permission. To login to system user follow by below steps:

         Open Archive site. The login screen will display

Figure 2.1 Login view

         Enter all required and valid data.

         Click on Sign in button to access system.

         Main screen will display if user account is valid. System will display error and request re-enter data if user account invalid.

3.     Search View.

This feature allows user retrieve documents by Content Type, Advance Search or Global Search. This allow user save the search condition as template query.

a.      Content Type search View.

This feature provide quit search on Content Type. The returned results are all documents that contain in this Content Type.

         In Search screen, double-click on Content Type item in left hand side Content Type list.

         Search result view will display on right hand side.

Figure 3.4 Content Type Search

b.     Advance Search View.

This feature provide search pane with set of conditions. The search conditions are based on the Fields of selected Content Type. With provide search conditions, user can limit the number of returned search result.

         In search screen, user selects Content Type need to do advance search on the left hand side Content Type list.

         Open Advance Search pane by click on arrow button on top-right of search screen. The Advance Search pane will display to user enters conditions.

         Enter search condition to textboxes.

         Click on Add condition button to add more condition user want to filter. The additional condition will be added to condition set.

         Click on Reset conditions button to reset condition set to default.

         Click on Search button to run Advance Search.

         Click on Save query to save search conditions as template query.

         Click on Delete query to delete selected search query if it is already chose.

Figure 3.5 Advance search result.

Figure 3.5 Advance search result Add condition.

This feature allows user search all documents on all existed Content Type in system on Field values with key word.

         In search view, enter key word need to search to the global search textbox.

The importance way to input document is input document by scanner. To scan document to Archive system the scanner must support TWAIN driver and WIA driver.

To scan document to system we have two ways to do:

Connect to scanner by scan drop-down button

         Login to system with right permission.

         In Main button bar click on Capture button . Capture view will display.

         In Capture view click on Scan drop-down button .

         The popup menu will display. This menu consists of items that are list of Content Type. Login user just see Content Type that user has permission to access.

Figure 4.6 Scan popup menu.

         Select Content Type user want to scan document into.

         When the papers are empty in paper jam, system will confirm user continues scan or stop.

         Choose Continues to continue scanning.

         Choose No to stop scanning.

         The scanned document will be display on Thumbnail view and Content View.

To configure the number pages to break new document:

         In scan popup menu select Scan pages per document menu item. The submenu will display.

         Not set: Select this option to ignore setting.

         Select number of pages wants to break new document. The setting will effect when user scan document.

Connect scanner by Context menu.

         In capture view right-click on Default Batch on thumbnail view.

         The popup menu will open.

Figure 4.6 Scan popup menu

         Select Scan menu item. Submenu will open. This menu contain to menu item are:

o   Classify later: Choose this option to scan document immediately to the root batch. Those scanned pages called Loose pages. Those scanned pages will be classified after use finish scanning.

Figure 4.7 Scan type menu.

Figure 4.8 Content Type submenu

6.     Change Password.

This feature allows user change login password.

Figure 6.1 Change password menus

         Login to system with right permission.

         Click on Account button. Popup menu will display.

         Select on Change password menu item. Change password view will open.

Figure 6.2 Change password view.

         Enter required and valid data.

         Click on OK button to change password. Password will be change and system back to login screen request login again.

         Click on Cancel button to cancel change password action.
